# 数据可视化技能

自动分析数据并生成专业的可视化图表。

## 功能特性

### 支持的图表类型
- 📈 **折线图** - 时间序列数据、趋势分析
- 📊 **柱状图/条形图** - 分类数据比较
- 🥧 **饼图/环形图** - 占比和构成分析
- 🔵 **散点图** - 相关性分析
- 🗺️ **热力图** - 矩阵数据可视化
- 📦 **箱线图** - 数据分布和异常值检测
- 🌐 **地图可视化** - 地理数据展示

### 智能功能
- ✅ 自动识别数据类型和最佳图表类型
- ✅ 智能配色方案选择
- ✅ 自动格式化坐标轴和标签
- ✅ 支持交互式图表（Plotly）和静态图表（Matplotlib）
- ✅ 导出为 PNG、SVG、HTML 格式

## 使用方法

### 基础用法

**直接描述需求**：
- "将这些销售数据可视化为柱状图"
- "创建一个显示过去12个月趋势的折线图"
- "生成产品类别占比的饼图"

**自动选择图表类型**：
- "可视化这份数据"（AI 会分析数据特征并选择最佳图表）

### 高级用法

**自定义样式**：
- "创建深色主题的折线图"
- "使用蓝色系配色的柱状图"

**组合图表**：
- "创建一个包含折线图和柱状图的组合图"
- "生成2x2的子图网格显示多个指标"

**交互式图表**：
- "创建交互式散点图，可以缩放和悬停查看数据"
- "生成可以导出的 HTML 交互式仪表板"

## 工作流程

1. **数据分析** - 自动检测数据类型、分布和特征
2. **图表推荐** - 根据数据特征推荐最合适的可视化方式
3. **参数优化** - 自动设置标题、标签、图例、配色
4. **生成图表** - 使用 Matplotlib 或 Plotly 生成高质量图表
5. **导出保存** - 保存为指定格式

## 技术栈

**Python 库**：
- `matplotlib` - 静态图表生成
- `seaborn` - 统计可视化
- `plotly` - 交互式图表
- `pandas` - 数据处理

## 最佳实践

### 选择正确的图表类型
- **时间序列** → 折线图
- **分类比较** → 柱状图/条形图
- **占比构成** → 饼图/环形图
- **相关性** → 散点图
- **分布** → 直方图/箱线图
- **热度矩阵** → 热力图

### 提高可读性
- 使用清晰的标题和标签
- 避免过多的颜色和元素
- 确保文字大小适中
- 添加数据标签（当数据点不多时）
- 使用图例说明

### 配色建议
- 使用色盲友好的配色方案
- 保持配色一致性
- 对比色用于突出重点
- 避免使用过于鲜艳的颜色

## 示例场景

### 场景 1：销售数据分析
```
输入：CSV 文件包含日期和销售额
输出：折线图显示销售趋势，柱状图显示月度对比
```

### 场景 2：市场份额分析
```
输入：各公司的市场占比数据
输出：饼图显示市场份额分布，带百分比标签
```

### 场景 3：相关性分析
```
输入：两个变量的数据
输出：散点图显示相关性，包含回归线和相关系数
```

## 依赖安装

```bash
pip install matplotlib seaborn plotly pandas numpy
```

## 注意事项

- 大数据集建议使用采样或聚合以提高性能
- 交互式图表适合探索，静态图表适合报告
- 确保数据清洗完成后再进行可视化
- 考虑目标受众选择合适的复杂度

## 输出格式

- **PNG** - 高分辨率静态图片（适合插入文档）
- **SVG** - 矢量图形（适合印刷和缩放）
- **HTML** - 交互式网页图表
- **Jupyter Notebook** - 在笔记本中直接显示
