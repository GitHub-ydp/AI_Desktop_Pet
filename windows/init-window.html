配置 AI 能力（可选）

填入 DeepSeek API Key 后，宠物将具备真正的 AI 对话能力。

没有 API Key？可以先跳过，宠物会使用内置的默认回复。之后在设置 → API Key 管理 中随时添加。
如何获取 DeepSeek API Key？
  1. 访问 platform.deepseek.com，注册账号
  2. 进入「API Keys」页面，点击「创建 API Key」
  3. 复制 sk-... 开头的密钥粘贴到上方输入框
新用户有免费赠送额度，够用很久~
